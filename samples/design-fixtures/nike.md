# Design System Inspired by Nike

## 1. Visual Theme

Athletic, high-contrast, editorial. Black-and-white photography backgrounds
paired with oversized italic display type. CTAs are confident and
rectangular.

## 2. Color Palette & Roles

### Primary
- **Jet Black** (`#111111`): All text, primary page headlines.
- **Off White** (`#f5f5f5`): Page background, default canvas.

### Accent
- **Signal Orange** (`#fa5400`): Primary CTA, active states, brand accent.
- **Signal Orange Hover** (`#d64900`): CTA hover, accent hover state.

### Surface
- **Card Surface** (`#ffffff`): Card surface, elevated panel.
- **Hairline Gray** (`#d8d8d8`): Borders (default), dividers.

### Feedback
- **Danger Red** (`#d43f3a`): Error text.

## 3. Typography Rules

### Font Family
- **Headline**: `Nike Futura`, with fallbacks: `Futura, Impact, Arial Black, sans-serif`
- **Body**: `Helvetica Neue`, with fallback: `Helvetica, Arial, sans-serif`
- **Monospace**: `Menlo`, with fallback: `Consolas, monospace`
