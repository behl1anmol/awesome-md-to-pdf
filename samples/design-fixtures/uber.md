# Design System Inspired by Uber

## 1. Visual Theme

Utility-first and literally monochrome. Buttons are rectangles with
tight letter-spacing. Motion is minimal. Chrome never competes with
content.

## 2. Color Palette & Roles

### Primary
- **Pure Black** (`#000000`): All text, all buttons, primary CTA.
- **Pure White** (`#ffffff`): Page background, card surfaces.

### Secondary
- **Mid Gray** (`#545454`): Secondary text.
- **Stone Gray** (`#8c8c8c`): Tertiary text, captions, metadata.

### Surface
- **Light Panel** (`#f6f6f6`): Section background, elevated surface.
- **Border Default** (`#e2e2e2`): Borders, dividers.

## 3. Typography Rules

### Font Family
- **Display / Hero**: `UberMove`, with fallbacks: `-apple-system, Arial, sans-serif`
- **Body / UI**: `UberMoveText`, with fallback: `-apple-system, Arial, sans-serif`
- **Monospace / Code**: `UberMono`, with fallback: `Menlo, monospace`
