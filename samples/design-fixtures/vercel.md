# Design System Inspired by Vercel

## 1. Visual Theme

An ultra-clean developer aesthetic: pure white canvas, true black ink,
and a single accent for CTAs. Typography is geometric sans with tight
tracking.

## 2. Color Palette & Roles

### Primary
- **Pure White** (`#ffffff`): Page background, card surfaces.
- **True Black** (`#000000`): All text, primary CTA, borders (default).

### Accent
- **Link Blue** (`#0070f3`): Primary CTA, links, focus ring.
- **Hover Blue** (`#0761d1`): CTA hover, link hover state.

### Feedback
- **Error Red** (`#ee0000`): Destructive actions, error text.

## 3. Typography Rules

### Font Family
- **Primary**: Geist, with fallbacks: -apple-system, BlinkMacSystemFont, Segoe UI, sans-serif
- **Monospace**: `Geist Mono`, with fallback: `Menlo, Consolas, monospace`
