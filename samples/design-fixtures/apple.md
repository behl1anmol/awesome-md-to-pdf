# Design System Inspired by Apple

## 1. Visual Theme

Luxurious and product-forward. Full-bleed photography against pure white,
with microscopic amounts of near-black for text. Every pixel either
whispers or is completely silent.

## 2. Color Palette & Roles

### Primary
- **Pure White** (`#ffffff`): All backgrounds, primary page canvas.
- **Pure Black** (`#000000`): All text, primary headlines and body.

### Surface
- **Soft Gray 6%** (`rgba(0, 0, 0, 0.06)`): Card surface, subtle elevated panels.
- **Hairline Tint** (`rgba(0, 0, 0, 0.10)`): Borders (default), dividers, subtle separators.

### Accent
- **Link Blue** (`#0066cc`): Links, primary CTA.

## 3. Typography Rules

### Font Family
- **Display**: `SF Pro Display`, with fallback: `-apple-system, BlinkMacSystemFont, sans-serif`
- **Body**: `SF Pro Text`, with fallback: `-apple-system, sans-serif`
- **Monospace**: `SF Mono`, with fallback: `Menlo, monospace`
