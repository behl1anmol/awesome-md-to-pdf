# Design System Inspired by Stripe

## 1. Visual Theme

Technical trust: soft gradients behind confident sans-serif headlines,
generous whitespace, and a saturated accent used sparingly. The UI reads
like financial documentation rendered as a product.

## 2. Color Palette & Roles

### Primary
- **Ink** (`#0a2540`): All text, primary headings.
- **Mid Gray** (`#425466`): Secondary text.
- **Cloud Gray** (`#8898aa`): Tertiary text, metadata, captions.

### Surface
- **Paper White** (`#ffffff`): Page background, primary card surface.
- **Cool White** (`#f6f9fc`): Section background, elevated surface.
- **Border Default** (`#e3e8ee`): Borders, dividers.

### Accent
- **Indigo** (`#635bff`): Primary CTA, accent, active states, links.
- **Indigo Hover** (`#5851e5`): CTA hover, accent hover state.

### Feedback
- **Crimson** (`#df1b41`): Error text.

## 3. Typography Rules

### Font Family
- **Headline**: `Camphor`, with fallbacks: `Inter, system-ui, sans-serif`
- **Body**: `Sohne`, with fallback: `Inter, system-ui, sans-serif`
- **Monospace**: `Source Code Pro`, with fallback: `Menlo, monospace`
